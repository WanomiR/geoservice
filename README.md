# Geoservice
Case study project for GoKata Academy

## Instructions
Start all services and generate swagger docs with:
```bash
make all
```

Start services without generating swagger docs with:
```bash
make build
```

> [!note]
> It is imperative to use `make` as it loads environment variables.
